## K-Level Cognitive Taxonomy

| K-Level | Bloom's Level | Exam Implication |
|---------|--------------|-----------------|
| K1 | Remember | Recall a fact or definition |
| K2 | Understand | Explain, classify, summarize, differentiate |
| K3 | Apply | Use a technique in a given situation |
| K4 | Analyze | Break down a situation and select/evaluate approaches |

**CTAL-TA distribution**:
- K2: Most foundational rules (role, environment, oracle, quality characteristics)
- K3: Technique application (domain, state transition, scenario, decision table, metamorphic, keyword)
- K4: Complex analysis (technique selection, regression scope, test result analysis)
